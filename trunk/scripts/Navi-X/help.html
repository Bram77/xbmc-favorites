Navi-X v2.x help

 

Navi-X is a media browser which allows you to browse and play video files, audio files, show images and shows text files. Navi-X can be compared with a web browser but it uses PLX playlists files instead of HTML files. Like web pages PLX-files can be located anywhere on the internet but they can also be located locally on your Xbox.

 

Navi-X allows people to create their own playlists without having to know how to develop a Python script. PLX Playlist are simple text files and can be edited with any text editor like notepad (see below for the formatting). Playlist can also link to other playlist which is similar to HTML files.

 

Navi-X allows people to share and manage their own playlists. For example a user can develop a playlist and upload it to a HTML server. Many people already have a personal website. Just upload the playlist to your website and share the link with others. Changes made to your playlists will be visible to others immediately.

 

(PLX) Playlist file layout

 

version=7

#

#playlist properties:

background=[default, previous, <URL>]

logo=[none, previous, <URL>]

title=[<string>]

player=[mplayer, dvdplayer]

#

#next the playlist entries:

type=[playlist, video, audio, image, text, rss]

name=[<string>]

thumb=[default, <URL>]

URL=[<string>]

player=[mplayer, dvdplayer]

 

Playlist properties:

Version

Indicates the playlist version. Shall be ‘7’ in Navi-X 2.x

Background

Background image.

default uses the default Navi-X background image.

previous uses the image of the previous playlist. This setting is useful when a tree of playlists share the same background image. In this case only the root playlist image needs to be set.

<url> is a string to a background image.

Logo

Logo displayed at left of the screen.

none indicates no logo image.

previous is image of the previous playlist.

<url> is a string to a background image.

Title

Playlist Title string displayed at the top center of the screen.

Player (optional)

XBMC player to use. Can be used to overwrite the default player

mplayer use the XBMC internal mplayer to play all items in the playlist

dvdplayer use the XBMC internal dvdplayer to play all items in this playlist

Playlist entry:

Type

Indicate the type of media. Used to display the default thumb and to select the correct player.

playlists indicates the URL points to a linked playlist file.

video indicates the URL points to a video file or stream. The xbmc,Player is used to play this media.

audio indicates the URL points to an audio file or stream. The xbmc,Player is used to play this media.

image indicates the URL points to an image (jpg, gif, png, etc.). The xbmc.slideshow is used to display this media

text indicates the URL points to a plain text file. The Navi-X internal text viewer is used to display this media.

rss indicate the URL points to a RSS 2.0 feed XML file.  Used for podcasts.

Name

Entry name. Displayed in the list.

Thumb

Indicates the thumb image for this item

default indicates the default Navi-X media image is used

<url> is a string to a thumb image (normally around 100x100 pixels)

URL

URL string pointing to the media.

Player (optional)

XBMC player to use. Can be used to overwrite the default player

mplayer use the XBMC internal mplayer to play the entry

dvdplayer use the XBMC internal dvdplayer to play the entry

General:

#

Indicates a comment line

 

 

 

Note: In Navi-X you can view the playlist source using the ‘view playlist source’ menu option. Press the white button on your Xbox remote to show the context menu. Have a look at existing playlists and use them as example for your personal new playlists.

 

Note about URLs and local files:

URLs can point to files on the internet or local disk. For example:

URL=http://www..... (Points to a file on the internet)

URL=images\file.jpg (relative path. Points to a file in the “Navi-X\images” sub-directory

URL=Q:\images\file.jpg (absolute path)

 

What about Youtube?

Navi-X also includes a  Youtube HTML parser. For this the following additional types have been defined:

  • html_youtube – Youtube HTML page
  • search_youtube – Search Youtube option

You can use these types in your personal PLX playlists as well.
