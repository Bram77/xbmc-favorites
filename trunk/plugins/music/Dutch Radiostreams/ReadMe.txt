Installation instructions

1. Extract the files to your harddrive
2. Upload the content to a folder on your Xbox (for example 'Music' on your E or F drive.)
3. Set your "Unknown type cache - Internet" at 256 kb
4. Go to Music on the homescreen and select 'Add Location'
5. Select the folder you placed the content in.



Known issues


Problem: The folder 'Radio' doesn't have a thumbnail because of a bug that gives all the content into 
this folder the same thumbnail. 

Solution: Let XBMC cache the thumbnails from all radiostations first. 
Then rename 'Radio_read the readme first!.tbn' to 'Radio.tbn'

Note that you'll have to do this trick again when you remove the UserData folder.
